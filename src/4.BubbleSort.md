# Bubble Sort

# Description :

- Time Complexity :-
  O(n²) in the average and worst cases – and O(n) in the best case

### Code :

```
function solve(N,arr){
    for(var i=0;i<N;i++){
        for(var j=0;j<N-i-1;j++){
            if(arr[j]>arr[j+1]){
                swap(arr,j,j+1);
            }
        }
    }
    console.log(arr.join(" "));
}

function swap(arr,i,j){
    var temp=arr[i];
    arr[i]=arr[j];
    arr[j]=temp;
}

```
